# CleanShip (整理飞船)

这是一个功能强大的飞船物资管理插件。它允许玩家精确定义每一件废品或工具在飞船上的位置，从此告别混乱的船舱，让物资摆放整齐划一。  

A powerful tool to manage and organize items on your ship. Define precise coordinates for every scrap or tool and keep your ship cabin perfectly tidy.  

---

## ✨ 核心功能 (Features)

- **📍 自定义坐标 (Custom Coordinates)**：支持为每种物品单独设置理想的坐标位置。  
- **⌨️ 手动微调 (Manual Editing)**：通过 UI 面板直接输入数值，精准控制物品的 XYZ 轴位置。  
- **🔍 智能搜索列表 (Item Search List)**：提供飞船上已有物品的实时列表，支持搜索，快速定位你想整理的物资。  
- **🌈 状态区分 (Visual Status)**：在物品列表中，已自定义坐标的物品将以**不同颜色**高亮显示，方便区分。  
- **💾 配置持久化 (Persistent Config)**：坐标信息会自动保存到配置文件中，只需设置一次，永久生效。  
- **🎯 灵活整理 (Selective Organization)**：提供“仅整理自定义物品”选项，避免干扰你不想移动的物资。
- **📐 UI 与字体调节 (Adjustable UI & Font)**：支持自定义 UI 面板和字体大小。

---

## 🎮 使用说明 (How to Use)

1. **打开 UI 面板**：在游戏内按下键盘上的 `**=**` 号键。
2. **设置方法**：
  - 方法一：手持需要自定义坐标的物品，然后点击‘记录手持’。  
    - 方法二：‘飞船内物品扫描’下会显示飞船里的物品，点击右边的设为脚下即可
3. **快捷整理**：点击面板上的整理按钮，所有匹配的物品会自动整理到设定位置。
